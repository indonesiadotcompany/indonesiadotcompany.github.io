Install solr server

Sep 27, 2017

Here are the steps for Install solr server - base of search engine for magento these steps run on ubuntu 16.04, because we follow this http://devdocs.magento.com/guides/m1x/other/ht_magento-solr.html, we must install jdk 7 from ppa, but ubuntu 16.04 default with jdk 8 add-apt-repository ppa:openjdk-r/ppaapt updateapt-get install openjdk-7-jdkapt install supervisorsu - usermkdir apps && cd appswget https://archive.apache.org/dist/lucene/solr/3.4.0/apache-solr-3.4.0.tgz<br />tar xzf ../apache-solr-3.4.0.tgzmv apache-solr-3.4.0 solr34cp /var/www/magento/lib/Apache/Solr/conf/* solr34/example/solr/conf/cat >> apps/solr34/example/logging.properties # Default global logging level:.level = INFO# Write to a file:handlers = java.util.logging.FileHandler# Write log messages in XML format:# Use java.util.logging.SimpleFormatter to log like Solr logs to the screen by defaultjava.util.logging.FileHandler.formatter = java.util.logging.XMLFormatter# Log to the current working directory, with log files named solrxxx.logjava.util.logging.FileHandler.pattern = /home/ehbm2/logs/solr%u.log[CTRL-C]exitcat >> /etc/supervisor/conf.d/solr.conf [program:solr]user=ehbm2#directory=/home/ehbm2/apps/solr141/exampledirectory=/home/ehbm2/apps/solr34/examplecommand=java -Djava.util.logging.config.file=logging.properties -jar start.jarautostart=trueautorestart=true[CTRL-C]/etc/init.d/supervisor restart
