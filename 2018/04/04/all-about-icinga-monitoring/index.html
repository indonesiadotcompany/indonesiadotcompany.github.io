added a line into the servicedefinition:
Code
  1. apply Service “ssh” {
  2. import “generic-service”
  3. check_command = “ssh”
  4. vars.port = host.vars.ssh_port
  5. assign where host.address && host.vars.os == “Linux”
  6. ignore where host.name == “localhost” /* for upgrade safety */
  7. }

and in the hostdef:
Code
  1. object Host “webarchiv” {
  2. …
  3. vars.ssh_port = 1023
  4. …
