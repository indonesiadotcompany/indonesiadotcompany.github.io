digitalocean DO cli

Mar 4, 2018

digitalocean snapshot and recreate based on that snapshot (clone like)

doctl compute droplet-action snapshot DropletID –snapshot-name SnapshotName
doctl compute droplet-action snapshot 83360000 –snapshot-name 20180303-birau-snap

doctl compute droplet create NewDropletName –size 512mb –image SnapshotID –region sgp1 –ssh-keys Ssh:key:hexadecimal

doctl compute droplet create birau-server-2 –size 512mb –image 32187000 –region sgp1 –ssh-keys b0:10:00:d6:8b:00:4f:96:39:1b:89:00:00:00:00:00

elasticsearch tips and tricks

Mar 4, 2018

elasticsearch list indexes

https://www.google.co.id/search?q=elasticsearch+list+indexes
https://www.elastic.co/guide/en/elasticsearch/reference/1.4/_list_all_indexes.html

curl ‘localhost:9200/_cat/indices?v’

elasticsearch installation

https://github.com/Smile-SA/elasticsuite/wiki/ServerConfig-2.x
https://www.digitalocean.com/community/tutorials/how-to-install-java-with-apt-get-on-ubuntu-16-04

elasticsearch config

cluster.name: namacluster
bootstrap.memory_lock: true

# http://devdocs.magento.com/guides/v2.1/config-guide/elasticsearch/es-overview.html#es-install-es
index.query.bool.max_clause_count: 102400

# this must be set on non public ip
network.host: 127.0.0.1
http.port: 9200
# https://www.elastic.co/guide/en/elasticsearch/reference/2.4/modules-threadpool.html
processor: 2
threadpool.bulk.queue_size: 4000

# https://github.com/Smile-SA/elasticsuite/wiki/ServerConfig#configure-elasticsearch
script.inline: on
script.indexed: on
