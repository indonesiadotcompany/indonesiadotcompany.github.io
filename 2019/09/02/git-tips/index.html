Git tips

Sep 2, 2019

git merge

Contohnya branch master isinya kurang lengkap, dan branch hotfix isinya lengkap lalu git merge untuk menggabungkan isi dari branch hotfix ke master

git checkout master
git merge hotfix

Referensi:

https://git-scm.com/book/en/v2/Git-Branching-Basic-Branching-and-Merging

git undo merge

Ini adalah langkah untuk mengembalikan data yang terlanjur di merge ke repository log yang diinginkan

git reflog
git reset --hard <id_repository_log>

Langkahnya, lihat dulu id repository log mana yang terakhir dilakukan, setelah itu baru dikembalikan
