how to stop docker that always restart

Aug 11, 2019

googling: https://www.google.com/search?q=docker+restart+always+stop

https://stackoverflow.com/questions/39072357/how-to-stop-a-docker-container-which-started-with-restart-always/39072476

You can update docker container.
use sudo docker update –restart=no  to update –restart flag of the container.
Now you can stop the container.
