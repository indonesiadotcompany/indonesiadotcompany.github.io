Ubuntu dns local cache for faster dns resolver

Aug 11, 2019

googling: https://www.google.com/search?q=ubuntu+dns+local+cache

https://superuser.com/questions/1291450/how-can-i-run-a-local-caching-dns-resolver-on-ubuntu-14-04

The easiest way is probably nscd.
Install nscd, using the default config.
Update /etc/nsswitch.conf and add cache before dns to the hosts line. (Edit: this part is not applicable to glibc as used in most linux distributions but is needed on fbsd)
hosts: files cache dns
then update /etc/resolv.conf at the beginning of the nameserver use 127.0.0.1
nameserver 127.0.0.1
